# Flask REST API - User Management

This project is a simple REST API built using Flask.

## Features
- GET all users
- GET single user
- POST create user
- PUT update user
- DELETE user

## Tech Used
- Python
- Flask

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python app.py

3. Test using Postman or Curl
